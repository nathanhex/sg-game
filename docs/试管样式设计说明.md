# 试管样式设计说明（待确认）

## 当前问题
- **试管占比太小**：宽度约 60px，在界面里显得小气。
- **底部太尖**：使用 `border-radius: 8px 8px 50% 50%`，底部是半圆形收口，视觉上像「尖底」。

## 设计方向

### 1. 放大试管占比
- **宽度**：由 60px 调整为 **80px**（桌面端），移动端用 clamp 适当放大。
- **单格高度**：由 36px 调整为 **40px**，整体试管更高、更易点、更易看清液体。
- 容器仍用 `repeat(auto-fit, minmax(...))`，试管数量多时自动换行，单屏试管会更大、更醒目。

### 2. 底部改为「圆底」、不尖
- **不再用** 底部 `50% 50%`（半圆尖底）。
- **改为** 固定像素圆角，让底部是一段**平缓的弧**，像真实试管的圆底：
  - 例如：`border-radius: 12px 12px 24px 24px`（上略小、下略大），或
  - 底部一段用稍大的统一圆角，如 `... 20px 20px`，视觉上「圆底」而不是「尖底」。
- 最底下一格液体（`.tube-cell:first-child`）的圆角与试管底部一致，贴合圆底。

### 3. 视觉风格（与现有一致）
- 保持童话糖果风：玻璃高光、淡紫边框、内阴影。
- 液体仍为格状填充，顶部一格有「液面」高光。
