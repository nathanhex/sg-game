# 游戏索引首页设计

## 目标

- 站点根路径 `/` 为**游戏索引页**，卡片式布局，延续糖果梦境风格。
- 试管倒水为第一个游戏，迁入子路径 `tube-pour/`，后续游戏按同样方式放入各自子目录。
- **风格策略**：索引页统一用糖果梦境；各游戏可有独立风格，互不影响。

## 结构

```
/
  index.html       # 索引页（卡片列表）
  index.css        # 索引页样式
  tube-pour/       # 试管倒水
    index.html
    style.css
    game.js
    audio.js
    audio.css
    sounds/
```

## 索引页

- **风格**：与试管游戏统一的糖果梦境（粉蓝渐变、云朵与糖果山、Baloo 2 / Nunito、圆角卡片）。
- **内容**：标题 + 游戏卡片网格；每张卡片：图标/缩略、名称、简短描述、点击进入游戏。
- **动效**：页面加载时卡片错落出现（animation-delay），悬停轻微上浮与阴影。
- **扩展**：后续新游戏在索引页增加一张卡片，并新建对应子目录即可。

## 实现要点

- 索引页使用独立 `index.css`，复用糖果梦境色板与背景装饰类，不引用游戏内 `style.css`，避免样式冲突。
- 试管游戏所有资源置于 `tube-pour/`，相对路径不变，无需改 game.js / audio.js 逻辑。

## 添加新游戏（后续扩展）

1. 新建子目录 `<game-id>/`（如 `memory-match/`），与 `tube-pour/` 同级，放入该游戏的所有资源。
2. 在索引页 `index.html` 中增加一张游戏卡片，链接至 `<game-id>/`。
3. 游戏内部可自由定义样式和交互，与索引页及其他游戏隔离。
