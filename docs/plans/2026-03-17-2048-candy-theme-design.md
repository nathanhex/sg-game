# 2048 糖果风整体设计

**日期**: 2026-03-17  
**目标**: 让 2048 与索引页糖果梦境主题统一，解决背景与控件不协调、按钮布局混乱的问题。

---

## 1. 问题简述

- **背景与内容脱节**：页面已是粉蓝渐变+云朵+糖果山，但游戏区仍是原版 2048 的灰棕配色（棋盘 #bbada0、按钮 #8f7a66、文字 #776e65），风格冲突。
- **按钮布局混乱**：返回列表与标题、分数都用 float，above-game 里说明文字、夜间模式图标、新游戏/撤销分散左右，视觉层级不清晰。

---

## 2. 设计方向

- **调性**：与索引页一致——柔和、圆润、高饱和度糖果梦境，面向儿童/家庭。
- **原则**：整页视为「一张放在糖果背景上的游戏卡」——内容区用统一的设计语言（圆角、糖果色、清晰层级）。

---

## 3. 布局与层级

### 3.1 顶部栏（heading）

- **结构**：单行，从左到右顺序明确。
- **左**：`返回列表` 链接（保留现有糖果风 pill 按钮样式）。
- **中**：标题「2048」（可略放大、加粗，占视觉重心）。
- **右**：当前分数 + 历史最佳，两个分数块横向排列。
- **实现**：用 flexbox，`justify-content: space-between`，`align-items: center`；小屏可换行或缩小字号，避免重叠。

### 3.2 操作区（above-game）

- **结构**：一行，左侧说明、右侧操作组。
- **左**：游戏说明文案（保持一句简短介绍）。
- **右**：操作组横向排列——夜间模式切换 +「新游戏」+「撤销」，同一视觉权重，间距统一（如 8px）。
- **实现**：flexbox，`justify-content: space-between`，`align-items: center`；右侧用 flex + gap 排三个控件。

### 3.3 棋盘区（game-container）

- **位置**：居中，与上方留出适当间距。
- **外框**：整体视为「游戏卡」——圆角（如 16–20px）、浅色半透明或浅紫/白底、柔和阴影，与糖果背景融合。
- **内部**：棋盘底色改为糖果系（见下配色），格子与数字块统一圆角。

---

## 4. 配色（糖果梦境）

| 元素 | 当前 | 调整后 |
|------|------|--------|
| 页面背景 | 已为渐变+云+山 | 保持 |
| 主文字 | #776e65 | 与索引一致 #3D2D5C（--text-main） |
| 标题/分数文字 | 同左 | 同左，分数块内可用白色或浅色 |
| 分数块背景 | #bbada0 | 柔和紫/粉，如 #C8A8E8 / #E8D0F8，白字或 --text-main |
| 操作按钮（新游戏、撤销） | #8f7a66 | 与返回列表统一：白底+淡紫边框，或浅紫填充+深紫字 |
| 夜间模式按钮 | 图片 | 保持，可加圆角/边框与其它按钮统一 |
| 棋盘外框 | #bbada0 | 浅紫/薰衣草，如 #D4C0F0 或 rgba(212,192,240,0.9) |
| 空格子 | rgba(238,228,218,0.35) | 略偏糖果，如 rgba(255,255,255,0.5) 或极浅紫 |
| 数字块 2–2048 | 米黄→橙→深橙 | 糖果渐变：奶油→蜜桃→珊瑚→粉→紫（保持对比度，深色块白字） |
| 超大数字块 | #3c3a33 | 深紫 #3D2D5C 或同系深色 |

---

## 5. 字体与圆角

- **字体**：与索引一致，`"Baloo 2", "Nunito", "Microsoft YaHei", sans-serif`。
- **圆角**：按钮/分数块 12–14px；棋盘 16–20px；格子与数字块 10–12px。

---

## 6. 弹窗与状态

- **游戏结束/胜利/重新开始确认**：遮罩与弹框背景改为糖果系（如半透明紫、白底圆角），按钮与主操作区风格一致。

---

## 7. 夜间模式

- 保持现有逻辑：切换时 body/html 背景改为深色；关闭时清除 inline 样式以恢复 CSS 渐变。
- 可选后续：夜间模式也提供「深紫糖果」渐变，与日间形成同一家族。

---

## 8. 实现范围

- **文件**：`games/2048/index.html`（必要时微调 DOM 顺序或包裹）、`games/2048/style/main.css`（布局、配色、圆角、字体全覆盖）。
- **不动**：现有 JS 逻辑（含夜间模式、计分、弹窗显示），仅通过 class 与现有结构保持兼容。
- **响应式**：保留现有 520px 断点，在媒体查询内同步应用新配色与圆角，布局用 flex 自然适配。

---

## 9. 验收要点

- 整页视觉与索引页糖果风统一，无灰棕「原版 2048」残留。
- 顶部栏、操作区、棋盘层级清晰，按钮与链接排列整齐、易点。
- 分数、按钮、棋盘、数字块均使用糖果配色与统一圆角/字体。
