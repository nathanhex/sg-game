# sg-game

儿童向小游戏合集。打开 `index.html` 选择游戏，或直接进入各游戏子目录。

## 游戏列表

- **试管倒水** (`games/tube-pour/`) — 把同色液体倒进同一根试管

## 项目结构

```
/
  index.html    # 游戏索引页（糖果梦境风格）
  index.css     # 索引页样式
  games/        # 游戏子目录
    tube-pour/  # 试管倒水
  docs/         # 设计文档（含 PROJECT-STRUCTURE.md 目录规范）
```

## 音效

试管倒水音效可选：打开 `games/tube-pour/tools/generate-sounds.html` 生成 WAV，放入 `games/tube-pour/sounds/` 即可。
